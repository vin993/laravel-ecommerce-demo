# Laravel Demo Project

This is a sanitized, generic Laravel project prepared for public sharing and portfolio demonstration. All client-specific data, secrets, and branding have been removed.

## Features
- Modern Laravel application structure
- Example service integrations (API, email, cache, etc.)
- Modular, clean codebase
- Ready for customization and extension

## Setup
1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/your-laravel-demo.git
   cd your-laravel-demo
   ```
2. **Install dependencies:**
   ```sh
   composer install
   npm install
   ```
3. **Copy and configure environment:**
   ```sh
   cp .env.example .env
   # Edit .env with your own values
   php artisan key:generate
   ```
4. **Run migrations and seeders (if any):**
   ```sh
   php artisan migrate --seed
   ```
5. **Start the development server:**
   ```sh
   php artisan serve
   ```

## Folder Structure
- `app/` - Application code (controllers, models, services)
- `config/` - Configuration files (all secrets via .env)
- `database/` - Migrations and seeders
- `public/` - Web root
- `resources/` - Views and assets
- `routes/` - Route definitions
- `scripts/` - Utility scripts (sanitized)
- `storage/` - Logs, cache, and uploads (gitignored)
- `tests/` - Test suite

## Security & Privacy
- No secrets, API keys, or client data are present
- All sensitive config is in `.env.example` as placeholders
- All business/client logic and branding have been removed or generalized

## Tech Stack
- Laravel (PHP)
- Composer
- Node.js & NPM (for frontend assets)

## Contributing
Feel free to fork and use as a starter for your own projects!

# Introduction

Bagisto is an opensource [laravel eCommerce](https://www.bagisto.com/) framework built on some of the hottest technologies such as [Laravel](https://laravel.com/) (a [PHP](https://secure.php.net/) framework) and [Vue.js](https://vuejs.org/) a progressive Javascript framework.

Bagisto can help you cut down your time, cost, and workforce for building online stores or migrating from physical stores to the ever-demanding online world. Your business—whether small or huge—can benefit. The best part, it's straightforward to set it up!
